Projet IA : Reconnaissance de fleurs

***** Première étape : identifier et définir le problème *****

On donne une image de fleur à notre IA et elle nous retourne le nom (espèce) de cette fleur.
C'est une classification: il s'agit de trouver à quelle catégorie appartient l'image (= à quelle espèce appartient la plante).

***** Deuxième étape : comprendre les données *****

Quantité:

Nous disposons d'une base de données composée d'une centaine de fleurs différentes. Chacune des fleurs à une grande quantité d'image de type jpeg.

Qualité:

Les données sont diverses: angles de vue et couleurs variées pour chaque espèce, distance de la prise variable, états d'éclosion différents.
Sur certaines images les fleurs ne sont pas les éléments centraux, elles ne sont visibles que sur une partie. Cela rajoute un certain défi pour l'IA.
